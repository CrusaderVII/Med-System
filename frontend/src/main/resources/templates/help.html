НАЗАД

- Начало эксплуатации системы

Для начала эксплуатации системы необходимо выполнить вход в систему, введя логин и пароль, выданный пользователю доверенным лицом.

- Главное меню

В гланом меню имеются опции РАБОТА, АРХИВ, ПОМОЩЬ и ОБРАТНАЯ СВЯЗЬ.

- Опция РАБОТА

Опция предоставляет возможность использовать данные о пациентах:
  • Ввести данные нового пациента
  • Получить данные о пациенте (по его ФИО)
  • Получить данные о пациенте с нарушение моторных функций
  • Получить данные о рекомендациях для пациента (по его ФИО)

- Опция АРХИВ

Опция позволяет просматривать данные о пациентах, диагнозах, ТРС и уровнях GMFCS.

- Опция ПОМОЩЬ

Опция предоставляет информацию о функционале системы.

- Опция ОБРАТНАЯ СВЯЗЬ

Опция предоставляет данные о возможностях связаться с разработчиком системы.
